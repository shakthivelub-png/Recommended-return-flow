Recommended Web Return Flow

Aligned with existing replacement UX. No UI redesign.

1. Initiate Return

User starts return from Order Details page.

2. Eligibility Check

System validates return eligibility.

3. Recommended Next Step

System highlights the best next action (refund / replacement).

4. Confirmation

User confirms recommended action.

5. Completion

Return is initiated and status is updated.

Documented for alignment & next-step discussion.
